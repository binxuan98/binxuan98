## Selected Work / 代表项目

| Project | Description |
| --- | --- |
| [**ChineseLandscapeBench-12**](https://github.com/binxuan98/Chinese-landscape-painting-dataset) | 面向中国山水画生成、风格分析与图像评估的 9,600 张图像数据集，包含 12 个均衡类别及完整的元数据、质量统计和复现工具。 |
| [**Deep Learning Visual Lab**](https://github.com/binxuan98/Deep-Learning-Visual-Lab-Platform) | 面向人工智能课程的本地可部署实验平台，覆盖目标检测、图像分割、分类、增强、视频分析与风格化。 |
| [**AI + IoT Monitor System**](https://github.com/binxuan98/ai-iot-monitor-system) | 面向项目化教学的智慧环境监测与异常预警平台，贯通数据采集、分析、可视化、测试和团队协作。 |
| [**Codex Textbook Writer Skill**](https://github.com/binxuan98/codex-textbook-writer-skill) | 面向职业教育与高等教育教师的教材编写工作流，覆盖资料核验、教学设计、章节写作、DOCX 统稿与交付 QA。 |
